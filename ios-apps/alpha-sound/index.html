Alpha Sound

Alpha Sound is a simple ABC’s learning program for children. Children learn their letters by seeing and hearing the letter names.

Features

  • Hear the name of the letter
  • Uppercase and lowercase letters
  • Quickly access any letter
  • Autoplay letters
  • Large easy to read letters
  • Includes numbers 0 thru 10
