RubyMotion

RubyMotion is a toolchain that lets you develop iOS using Ruby. I’m a big fan of Ruby and have been using it on and off since 2001. It’s been a huge timesaver in many cases allowing me to quickly script solutions. When I heard it was coming to iOS I had to check it out. The toolchain and the way you use the various components matches the rails development process. Open your favorite text editor and a terminal, and you’re ready to go.
